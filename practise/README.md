# Privacy Preserving Multiple Linear Regression
##  RUN:
1. `make clean` to clean the compiled file
2. `make` to compile the code
3. `.\party` to run the party
4. `./client1 ../data/Student_Performance_client1.csv 5000` to run the client 1 with it's csv data file at port 5000.
5. `./client1 ../data/Student_Performance_client2.csv 5001` to run the client 2 with it's csv data file at port 5001.
